# A simple Sudoku Game


## To use this app

Visit the deployed Github Page [Sudoku Game](https://ssmeds.github.io/sudoku-game/)


If you clone he code run the following to see the local version

### `npm start`
### `npm run build`


## Tech Stack

- React JS
- Redux Toolkit
- SCSS
- Node js
